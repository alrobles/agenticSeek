## What EcoSeek Changes

EcoSeek adapts the AgenticSeek agent runtime for scientific and ecological
workflows. Changes include:

- Integration with [AgenticPlug](https://github.com/alrobles/agenticplug)
  as the secure gateway and connector layer.
- Support for [EcoCoder](https://github.com/alrobles/ecocoder) and
  [EcoAgent](https://github.com/alrobles/ecoagent) as domain-specialized
  intelligence and tool backends.
- DeepSeek BYOK (Bring Your Own Key) provider path for users who lack
  local GPU hardware.
- Scientific workflow tooling for ecological and computational biology
  use cases.
